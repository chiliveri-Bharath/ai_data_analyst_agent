# ai_data_analyst_agent
# 🤖 AI Data Analyst Agent

An AI-powered data analysis tool that allows users to upload datasets and interact with them using natural language queries.

---

## 🚀 Features
📊 Automated data cleaning (handling missing values,duplicates, preprocessing)
📈 Smart visualization with auto chart selection
🧠 Natural Language Query support (ask questions about your data)
⚡ Fast responses using Groq API (LLM integration)

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Streamlit
* Groq API
* Plotly / Matplotlib

---

## 📌 Use Cases

* Business data analysis
* Exploratory Data Analysis (EDA)
* Quick insights without coding
* Academic projects

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/chiliveri-Bharath/ai-data-analyst-agent.git
```

2. Navigate to project:

```
cd ai-data-analyst-agent
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
